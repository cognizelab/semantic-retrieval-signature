# Word-Pair Stimulus

`Word-lists.xlsx` contains the word pairs corresponding to trials in the 3-AFC
semantic retrieval task.

In the manuscript analyses, semantic distance was quantified for each
probe-target pair as `1 - cosine similarity` between static word vectors. The
best behavioural fit was obtained with ConceptNet Numberbatch v19.08 sparse
word vectors retaining the strongest 48% of weights.

Distractor-related semantic distances were also checked to confirm that
distractor similarity did not systematically covary with the probe-target
semantic distance of interest.

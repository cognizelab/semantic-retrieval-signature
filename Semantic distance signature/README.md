# Semantic Distance Signature Maps

This directory contains group-level neural signature maps derived from the
semantic distance predictive models.

## Surface

`Surface/` stores CIFTI dense scalar files in fsLR 32k surface space.

### T-PLS

The T-PLS maps correspond to the main predictive model in the manuscript.

- `z_stat_predictive_weight_map_semantic_retrieval_signature.dscalar.nii`:
  bootstrapped predictive-weight z-statistic map. This is the primary semantic
  distance signature map for applying the signature to other samples, for
  example by dot product.
- `z_stat_activation_map_semantic_retrieval_signature.dscalar.nii`:
  bootstrapped Haufe-transformed activation z-statistic map.
- `mask_weight_FDR05.dscalar.nii`: predictive-weight mask surviving FDR q < 0.05.
- `mask_activation_FDR05.dscalar.nii`: activation mask surviving FDR q < 0.05.
- `mask_overlap_FDR05.dscalar.nii`: overlap between weight and activation masks;
  this overlap defines the primary ROIs used for follow-up analyses.

### SVR

The SVR maps provide the supplementary support-vector-regression model.

- `z_stat_predictive_weight_map_semantic_retrieval_signature.dscalar.nii`:
  bootstrapped predictive-weight z-statistic map.
- `z_stat_activation_map_semantic_retrieval_signature.dscalar.nii`:
  bootstrapped Haufe-transformed activation z-statistic map.
- `mask_weight_FDR05.dscalar.nii`: predictive-weight mask surviving FDR q < 0.05.
- `mask_activation_FDR05.dscalar.nii`: activation mask surviving FDR q < 0.05.

## Volume

`Volume/` stores NIFTI files for the T-PLS semantic distance signature in
volumetric space.

- `z_stat_predictive_weight_map_semantic_retrieval_signature.nii`: volumetric
  predictive-weight z-statistic map.
- `z_stat_predictive_weight_map_semantic_retrieval_signature_thresholded_FDR05.nii`:
  FDR-thresholded volumetric predictive-weight map.

## Notes

Surface and volumetric signatures were generated by applying the same T-PLS
model parameters separately in surface and volume spaces. They are separate
spatial implementations rather than spatial transformations of one another.
